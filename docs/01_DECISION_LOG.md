# Decision Log v0.1

## D000｜项目用途边界

- 决策：本项目仅作为研究 / 工程分析工具，不作为医疗诊断软件。
- 影响：GUI、CLI、文档、输出结果不得给出医疗诊断结论。
- 决策人：Owner。
- 状态：已确认。

## D001｜输入数据来源与字段

- 决策：当前使用 BIDMC CSV。
- 时间戳字段：`Time [s]`，单位秒。
- ECG 字段：` II`，字段名可能包含空格。
- PPG 字段：`PLETH` / ` PLETH`，字段名可能包含空格。
- 原始采样率：125 Hz，固定。
- 决策人：Owner。
- 状态：已确认。

## D002｜降采样策略

- 决策：ECG、PPG、时间戳同步降采样到 50 Hz。
- 决策人：Owner。
- 状态：已确认。

## D003｜R 峰检测时机

- 决策：R 峰优先在 125 Hz 原始 ECG 上检测，再映射到 50 Hz 输出。
- 原因：保留更高时间分辨率。
- 决策人：Owner。
- 状态：已确认。

## D004｜第三方库策略

- 决策：允许使用第三方 ECG R 峰 / QRS 检测库。
- 决策：允许 Codex 安装 pip 依赖。
- 决策：无 license 限制，不要求商业闭源。
- 决策：不保留自研 fallback。
- 决策：在算法实现前必须先做 M2a 第三方库评估与选型。
- 决策人：Owner。
- 状态：已确认。
