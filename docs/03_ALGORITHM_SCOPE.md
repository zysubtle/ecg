# Algorithm Scope v0.1

## 当前算法范围

本项目当前算法范围包括：

1. 读取 BIDMC CSV；
2. 识别时间戳、ECG、PPG 字段；
3. 对 ECG、PPG、时间戳同步降采样到 50 Hz；
4. 在 125 Hz 原始 ECG 上检测 R 峰；
5. 将 R 峰时间映射到 50 Hz 输出采样点；
6. 计算相邻 R 峰间隔 `rr_ms`；
7. 导出工程分析用 CSV；
8. 在最小 GUI 中显示 ECG 波形和 R 峰。

## 当前不做

- 医疗诊断；
- 临床级准确性声明；
- HRV 指标；
- PPG IBI；
- 多导联融合；
- 自研 R 峰检测 fallback；
- 复杂噪声质量分级；
- 批量处理；
- 人工修正 R 峰。

## R 峰检测策略

已确认：

- 不默认从零自研；
- 先做 M2a 第三方库评估；
- M2a 后由 Owner 决策主 detector；
- 不保留自研 fallback；
- detector 必须通过 adapter 封装。

## 异常数据处理边界

当前仅要求记录风险和基本工程合理性检查，不要求完整质量分级体系。

后续可能需要处理：

- 噪声；
- 漏搏；
- 早搏；
- 信号中断；
- 时间戳非单调；
- 采样率偏离；
- 字段缺失；
- 输出结果空值。
